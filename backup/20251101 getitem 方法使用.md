## __getitem__方法
- 作用： 类对象可以像字典对象一样取值（dict["key"]）
- 用法：object["key"] 
  - 系统会自动调用getitem方法，返回return 值；
  - 不存在值，抛出异常
  - 这个过程中 类对象key  是否存在，都会调用__getitem__方法返回其规定值
- 编程:
```python
test = {
    'key1': 'value1',
    'key2': 'value2',
    'key3': 'value3',
    'key4': 'value4'
}
 
class Element:
    """获取元素"""
    def __init__(self):
        self.data = test
 
    def __getitem__(self, item):
        """获取属性"""
        data = self.data.get(item, '')
        if data:
            return data
        raise Exception('关键字不存在')
 
 
if __name__ == '__main__':
    print(Element()['key1'])
    print(Element()['key0'])
```
